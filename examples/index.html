User Feedback...

...is of enormous importance!
This plugin aims at making it extremely easy to integrate a nice-looking user feedback form onto your page.

Get started

How it works

The main goal is to keep it as simple as possible. Just link the required CSS and JavaScript file and initialize it in one line.

This will automatically use the default configuration and add a feedback button to the lower-left corner of your site.

By clicking the button, a modal dialog will be presented to the user, allowing him to specify his feedback.

For a demo, just click the feedback button on this page here.

Installation

Reference the stylesheet and JavaScript file. 

<head>
  ...
  <link rel="stylesheet" href="../jquery-bootstrap-feedback.css">  
  ...
</head>
<body>
  ...
  <script src="../jquery-bootstrap-feedback.js"></script>
</body>

Usage

The simplest way to use the plugin is to simply invoke 

$.feedback();

This hooks it up with its default configuration, adding a button at the lower-right corner of the page and displaying the default text.
